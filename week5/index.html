Chapter 10 Notes:

  • An exception is an error that produces a return value that can then be used by the program to deal with the error
  • Strict mode simply requires the following string to be added to the first line of a JavaScript file: 'use strict';
  • Linting tools are designed to highlight any sloppy programming practices or syntax errors, and will complain if certain style conventions are not followed.
  • An error object can be created by the host environment when an exception occurs, or it can be created in the code using a constructor function
  • Throwing an exception is just like Java.
  • Writing tests will help me catch more errors early and less time debugging later.
  • Test-driven development (TDD) is the process of writing tests before any actual code
