Notes for Week 2:

Chapter 2 Notes:

  • Use the key word "const" to declare a variable that will no be reassigned later in a program. Use key word "let" to declare a variable that could be changed later in the program.
  • "If the variable references a non-primitive data type, such as an array, function or object, then using const will not make it immutable. This means the underlying data inside the object can change (known as mutating the object)"
  • Template Literals are special strings that use ` to deliminate the string: `Hello`; This also allows us to add variables to strings: `Hello ${name}"
  • Symbols are a new primitive value that are used to create unique values to help avoid naming collisions. Use Symbol() to create these
