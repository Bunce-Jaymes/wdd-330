Chapter 5 Notes:

  • Object Methods "this". This key word always confuses me.
  • Methods can reference objects as "this". Functions can use "this" but it has no value until the function is called.
  • Arrow function do not have "this" it uses the outer normal function
  • When checking a property with ".hasOwnProperty" this will only return property belonging to that object where as "in object" will return true if the property has inherited it from another object.
  • IMPORTANT! Objects are copied by reference and changes to the copy or original will affect both.
  • Rules to JSON:
  • Property names must be double-quoted
  • Permitted values are double-quoted strings, numbers, true, false, null, arrays and objects
  • Functions are not permitted values
  • Best practice to pass parameters to date constructor like so: "new Date(year,month,day,hour,minutes,seconds,milliseconds)"
  • RegExp is " is a pattern that can be used to search strings for matches to the pattern. A common use is “find and replace” type operations."

Chapter 6 Notes:

  • document.querySelector() is one of the most useful item I will use. It allows you to grab items by CSS notation.
  • "document.querySelector('li:last-child')" allows to grab the last item.

Chapter 7 Notes:

  • " the event listener will let it know when the event happens, and the program can then respond appropriately."
  • Make sure you think carefully before using preventDefault() to change default behavior. Users will expect certain behaviors, and preventing them may cause confusion. Some events do not allow the default behavior to be prevented. This can vary from browser to browser, but each event object has a property called cancellable that returns false if it cannot be prevented. You can also see if the default behavior has been prevented by checking the defaultPrevented property.
